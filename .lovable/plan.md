

# Dharani Reddyvari — Data Engineering Portfolio

A bold, dark-themed single-page scroll portfolio optimized for US Data Engineering recruiters.

## Visual Foundation
- Dark charcoal/black theme with blue accent colors
- Clean Inter/system font typography with strong hierarchy
- Fully responsive layout, fast-loading, recruiter-friendly
- Reduced-motion media query support throughout

## Sections

### 1. Hero (Above the Fold)
- Name, title line, and professional summary
- CTA buttons: Download Resume, LinkedIn, GitHub, Email — with hover glow effects
- Location line with relocation openness
- Animated count-up stats row (25K–40K URLs/batch, ~300K SKUs/run, ~95% success target)
- Subtle animated background with slow-moving data-flow lines or network nodes

### 2. Experience (Animated Timeline)
- Vertical timeline with 3 roles (Dentsu Software Data Engineer, Dentsu Data Analyst, Roots Industries Intern)
- Cards reveal on scroll with staggered timing
- Expand/collapse accordion with smooth height transitions and rotating chevron
- Impact tags that animate in as colored chips
- Concise 4–6 bullet points per role with measurable outcomes

### 3. Case Study (Interactive Stepper)
- "LVMH Brand Monitoring Pipeline" — 6-stage interactive stepper
- Clicking a stage swaps content with fade + slide transitions
- Progress bar that advances with each step
- Stages: Inputs → Extraction → Transformation → Validation → Storage & Serving → Failure Handling
- "What I Owned" callout card that slides in on scroll

### 4. Projects (Card Grid)
- Animated project cards for Airbnb Spark Pipeline and Airflow Data Validation Pipeline
- Hover effects: lift, border glow, icon slide
- Expandable highlights accordion (3–5 bullets)
- GitHub and README link buttons with tech tags that stagger in

### 5. Skills (Filterable Grid)
- Category filter chips: Languages, Processing, Orchestration, Platforms, Databases, DevOps
- Smooth fade/scale transitions when toggling filters
- Active filter chip with animated underline/glow
- Hover micro-interactions on each skill card

### 6. Contact (Minimal CTA)
- "Open to US-based Data Engineering roles" headline
- Email, LinkedIn, GitHub, Download Resume buttons
- Slide + fade reveal animation with hover glow on buttons

### Navigation
- Sticky navbar with section links and active-section highlighting
- Smooth scroll behavior across all sections

### Placeholders
- Resume PDF URL, LinkedIn URL, GitHub URL clearly marked for easy replacement
